## よく利用するコマンド

| コマンド             | 内容                                                                             |
| -------------------- | -------------------------------------------------------------------------------- |
| bin/rails db:reset   | sqlite3 のテーブルを作り直して、シードファイルを読み込みダミーデータを作成する。 |
| bin/rails db:drop    | sqlite3 のテーブルをすべて削除する                                               |
| bin/rails db:migrate | sqlite3 のマイグレーションを実行する                                             |
